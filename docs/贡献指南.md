# 贡献指南

感谢您对智慧医疗系统项目的关注！

## 项目概述

智慧医疗系统是一个全栈医疗健康平台

## 快速开始

### 后端启动

```
cd src
mvn spring-boot:run
```

### 前端启动

```
cd smart-medicine-web
npm install
npm start
```

### Android 构建

```
cd smart-medicine-Android
./gradlew assembleDebug
```

## 提交规范

- feat: 新功能
- fix: 修复 Bug
- docs: 文档更新
- refactor: 重构

示例: `feat: 添加用户登录功能`
